# サンプル: The Simple Webservice — 出典と扱い

## 情報源

- **CDK コード**: `cdk-patterns/serverless`（MIT ライセンスの AWS サーバーレスパターン集）
  の `the-simple-webservice` パターン。
  - リポジトリ: <https://github.com/cdk-patterns/serverless>
  - パターン: <https://github.com/cdk-patterns/serverless/tree/main/the-simple-webservice>
- 構成は上記の **CDK コードのみ**（`lib/the-simple-webservice-stack.ts`）から導出した。
  同リポジトリが README で提供する構成図画像は参照していない。

## アイコンの扱い

`architecture-prod.svg` には **AWS 公式アイコンが埋め込まれている**。AWS はアーキテクチャ図
の作成目的でのアイコン利用を許諾しており、本サンプルはその範囲での完成した構成図である
（再利用可能なアイコン素材集としての配布ではない）。四半期更新でアイコンの見た目が変わる
ことがあるため、再生成時はバージョンを混在させない。出典・利用条件はスキルの
`references/icons.md` を参照。

非 AWS アイコン（`icons/browser.svg` / `icons/internet.svg`）は本スキルのオリジナル
（`icons/NOTICE.md`）。
